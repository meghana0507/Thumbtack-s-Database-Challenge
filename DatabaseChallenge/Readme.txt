Server(Back-end) Simple Database Challenge:

*Attacment includes two java files (Database.java and Main.java)

====How to run from command prompt=====
1. Unzip DatabaseChallenge.zip files
2. Navigate to DatabaseChallenge\src
3. run "javac Main.java" for compiling
4. run "java Main" for execution
5. Input commands to test the program
